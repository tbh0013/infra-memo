# Apacheアクセスログ解析メモ
サーバーのアクセスログから、**直近のアクセス上位IPを集計**するためのコマンドメモ  
トラフィック集中や不審アクセス調査の際に使用

---

## 直近のアクセス上位を確認する
**目的:**  
アクセスログ（`/var/log/httpd/access_log`）の中から、直近のアクセス N 件を対象に、  アクセス数が多い順に上位 IP アドレスを一覧表示する。

**使用例:**  
直近 **10,000件** のログから **上位30件** のアクセス元を表示。
```bash
tail -n 10000 /var/log/httpd/access_log \
  | awk '{print $1}' \
  | sort \
  | uniq -c \
  | sort -nr \
  | head -30
